# aframearjs
Raccolta di snippet di codice relativi ad A-Frame e AR.js. Usati in accoppiata, consentono di creare progetti di realtà aumentata direttamente nel browser.

**Attenzione!** Se si intende copiare e usare i file contenuti in questo repository, è necessario fare molta attenzione ai _path_ relativi a file esterni, librerie JavaScript linkate nella <head> del documento) e _assets_ vari.
